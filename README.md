# Flock Detector

An ESP32-based electronics project for learning embedded development, GPS integration, OLED displays, SD card logging, and wireless networking.

## Hardware

- ESP32-S3-devkitc-1
- SSD1306 OLED Display
- WS2812 RGB LED
- ATGM336H GPS Module
- microSD Card Module

## Goals

- Display GPS coordinates
- Log GPS data to an SD card
- Show system status on the OLED
- Use the RGB LED for status indication
- Learn Git and GitHub while developing the project

## Project Structure

- firmware/ – ESP32 source code
- hardware/ – Schematics, PCB files, and datasheets
- docs/ – Build notes and documentation
- images/ – Photos and screenshots
- logs/ – Example log files
